## Folder Structure:

- .gitignore
    - Tell git to ignore our .env file
- .env file
    - Not be upoaded to GitHub. This will house all our local connection variables

- create.sql
    - Table creation for us

- insert.sql
    - Insertion of data

- query.sql
    - All our sql queries

- demo.py
    - House a class that will run all of these for us at once.
